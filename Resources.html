Resources List

To help anyone seeking to learn more about resources for drawing, we decided to compile some highly requested information about them. As this is still a work in progress, stay tuned for more in the future!

What is the right drawing program for you?

Program Cost Pros Cons
Photoshop $9.99/month
  • A lot of tools and functions available
  • Can make a wide range of custom brushes
  • Pre-made brushes by Kyle Webster
  • A lot of editing filters
  • Rulers
  • Great exporting options: (CMYK) for print, save for web, etc
  • Certain effects exclusive to Photoshop
    		•
  • Large number of tools can be overwhelming for first time users
  • Cost adds up over time
  • Can occasionally freeze up (heavy duty program)
  • Not targeted for drawing specifically (more for photo editing)
    		•
    Paint Tool Sai $50 and includes Paint Tool Sai 2(beta)
  • You get Paint Tool Sai forever!
  • Access to Paint Tool Sai 2 (beta), which has MANY new functions
  • Geared toward drawing digitally
  • Intuitive to use
  • Great for 2D art
  • Light weight
  • Stabilizer (helps with smoothing)
    		•
  • More limited range of brushes (still customizable, but smaller library compared to Photoshop)
  • Less accurate lasso tool
  • Less filter options
  • Windows only
    		•
    Clip Studio Paint $50, as low as $25 on sale
  • Assets specifically geared toward creating comics digitally! (ex pre-made tones, panels, etc)
  • The EX version supports multiple pages
  • Many tutorials through cloud
    		•
  • Text tools are not the best (this issue may or may not have been fixed)
  • Large number of tools can be overwhelming for first time users
    		•
    Honorable mentions: ProCreate (Ipad only), Medibang (Free), Krita (Free), Fire Alpaca (Free)
    All in all, it doesn’t really matter which program you use! It all comes down to how much you understand your chosen software. While some features are only available for specific programs, there are usually ways to work around them. However, keep in mind that sometimes it’s best to use multiple programs for different effects, exporting options, etc, simply because some programs are just better at handling certain tasks. Sometimes, work arounds aren’t the most efficient or produce obviously inferior results.
